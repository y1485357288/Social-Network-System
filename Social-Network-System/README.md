# 社交网络图谱分析及智能推荐系统

## 📋 项目简介

本项目是一个功能完善的社交网络图谱分析及智能推荐系统，采用Java语言开发，实现了社交网络建模、人脉查询、社交距离计算、智能推荐和可视化等功能。

## 🚀 功能特色

### 核心功能
- **社交网络建模**: 基于邻接表实现图结构
- **人脉查询**: 支持一度、二度、多度人脉查询
- **社交距离计算**: 最短路径分析和社交距离计算
- **智能推荐**: 基于用户兴趣和共同好友的推荐算法
- **数据持久化**: CSV文件导入导出支持

### 扩展功能
- **图形可视化**: 可缩放拖拽的社交网络图谱
- **现代化界面**: 美观易用的图形用户界面
- **交互优化**: 数据加载进度提示和用户友好交互

## 🛠️ 技术栈

- **开发语言**: Java
- **GUI框架**: Java Swing
- **数据结构**: 自定义邻接表、哈希表
- **算法**: BFS、最短路径、推荐算法

## 🚀 快速开始

### 环境要求
- Java 8 或更高版本
- 至少 512MB 可用内存

### 运行方式

#### 方法1: 使用启动器（推荐）
```bash
java -cp bin com.socialnetwork.gui.Launcher
```

#### 方法2: 直接运行新版界面
```bash
java -cp bin com.socialnetwork.gui.SocialNetworkApp
```

#### 方法3: Windows批处理文件
双击 `run.bat` 文件

### 编译项目
```bash
javac -d bin src/com/socialnetwork/**/*.java
```

## 🎯 使用指南

### 1. 启动系统
运行启动器，选择"新版界面"

### 2. 加载数据
点击右上角"加载数据"按钮，选择数据文件

### 3. 选择用户
从下拉框中选择要分析的用户

### 4. 功能分析
- **一度人脉**: 查询直接好友
- **二度人脉**: 查询好友的好友
- **社交距离**: 计算与其他用户的距离
- **智能推荐**: 获取个性化推荐
- **可视化**: 查看网络图谱

### 5. 可视化操作
- **缩放**: 鼠标滚轮放大缩小
- **拖拽**: 按住鼠标左键移动图形
- **点击**: 点击节点查看用户详情

## 🔧 开发说明

### 核心类说明

#### AdjacencyList
社交网络图结构实现，支持节点和边的增删改查。

#### GraphAlgorithm
图算法实现，包含BFS遍历、最短路径计算等。

#### RecommendationEngine
智能推荐引擎，基于用户兴趣和社交关系进行推荐。

#### VisualizationPanel
图形可视化组件，支持缩放、拖拽和交互。

### 扩展开发
如需扩展功能，可参考现有代码结构：
- 添加新算法：在`core`包中创建新类
- 扩展界面：在`gui`包中继承或修改现有组件
- 数据源扩展：修改`DataLoader`支持新格式
