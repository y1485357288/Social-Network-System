# 社交网络系统使用说明

## 🚀 快速开始指南

### 第一步：环境准备
1. 确保已安装 **Java 8 或更高版本**
2. 下载项目文件到本地
3. 解压项目到合适目录

### 第二步：运行系统

#### 方法一：使用启动器（推荐）
```bash
java -cp bin com.socialnetwork.gui.Launcher
```

#### 方法二：直接运行新版界面
```bash
java -cp bin com.socialnetwork.gui.SocialNetworkApp
```

#### 方法三：Windows批处理文件
双击项目根目录下的 `run.bat` 文件

### 第三步：选择界面
启动后会出现界面选择器：
- **新版界面（推荐）**: 现代化设计，功能完整
- **旧版界面**: 原始界面，保留兼容性

## 🎯 功能使用详解

### 1. 数据加载

#### 首次使用
1. 系统启动后自动加载示例数据
2. 右上角显示"✓ 示例数据加载成功！"
3. 用户下拉框自动填充用户列表

#### 加载自定义数据
1. 点击右上角绿色"加载数据"按钮
2. 选择用户数据文件（users.csv格式）
3. 选择好友关系文件（friendships.csv格式）
4. 等待加载完成，显示统计信息

### 2. 用户选择
1. 从顶部下拉框选择要分析的用户
2. 系统自动更新用户信息和可视化中心
3. 结果区域显示当前用户基本信息

### 3. 功能分析

#### 一度人脉查询
- **功能**: 查询用户的直接好友
- **操作**: 点击左侧蓝色"一度人脉"按钮
- **结果**: 显示好友ID、姓名和共同兴趣

#### 二度人脉查询  
- **功能**: 查询好友的好友（间接关系）
- **操作**: 点击左侧紫色"二度人脉"按钮
- **结果**: 显示二度人脉列表和关联路径

#### 社交距离计算
- **功能**: 计算与其他用户的最短社交距离
- **操作**: 点击左侧黄色"社交距离"按钮
- **输入**: 在弹出的对话框中输入目标用户ID
- **结果**: 显示距离值和最短路径

#### 智能推荐
- **功能**: 基于兴趣和社交关系推荐用户
- **操作**: 点击左侧橙色"智能推荐"按钮
- **输入**: 输入推荐数量（默认5个）
- **结果**: 显示推荐用户列表和推荐理由

#### 多度人脉查询
- **功能**: 自定义查询深度的人脉关系
- **操作**: 点击左侧红色"多度人脉"按钮
- **输入**: 输入查询深度（1-5）
- **结果**: 显示指定深度的所有关联用户

#### 可视化更新
- **功能**: 刷新网络图谱显示
- **操作**: 点击左侧青色"可视化"按钮
- **效果**: 以当前用户为中心重新布局图形

### 4. 可视化操作

#### 缩放功能
- **放大**: 鼠标滚轮向前滚动
- **缩小**: 鼠标滚轮向后滚动
- **范围**: 支持0.5倍到3.0倍缩放

#### 拖拽移动
- **操作**: 按住鼠标左键拖动图形
- **效果**: 整个网络图谱跟随鼠标移动

#### 节点交互
- **查看详情**: 点击任意节点查看用户信息
- **颜色含义**:
  - **红色**: 当前中心用户
  - **蓝色**: 一度人脉（直接好友）
  - **绿色**: 二度人脉（好友的好友）
  - **灰色**: 其他关联用户

## 📊 数据文件格式要求

### 用户数据文件 (users.csv)
```csv
ID,姓名,兴趣1|兴趣2|兴趣3
1,张三,编程|音乐|运动
2,李四,阅读|旅游|美食
3,王五,游戏|电影|摄影
```

**格式说明**:
- 第一行必须是标题行
- ID为整数，姓名支持中文
- 兴趣使用竖线"|"分隔，最多3个
- 文件编码建议使用UTF-8

### 好友关系文件 (friendships.csv)
```csv
用户ID1,用户ID2
1,2
1,3
2,4
3,5
```

**格式说明**:
- 第一行必须是标题行
- 每行表示一对好友关系
- 关系是双向的，无需重复添加
- ID必须存在于用户文件中

## 🔧 常见问题解决

### 问题1: 程序无法启动
**可能原因**: Java环境未正确安装
**解决方案**: 
1. 检查Java版本：`java -version`
2. 如未安装，下载并安装Java 8+
3. 设置JAVA_HOME环境变量

### 问题2: 数据加载失败
**可能原因**: 文件格式错误或路径问题
**解决方案**:
1. 检查CSV文件格式是否符合要求
2. 确保文件编码为UTF-8
3. 检查文件路径是否包含中文或特殊字符

### 问题3: 可视化显示异常
**可能原因**: 图形组件渲染问题
**解决方案**:
1. 尝试调整窗口大小
2. 使用缩放功能查看完整图形
3. 点击"可视化"按钮刷新布局

### 问题4: 功能按钮无响应
**可能原因**: 未选择用户或数据未加载
**解决方案**:
1. 确保已成功加载数据
2. 从下拉框选择有效用户
3. 检查结果区域是否有错误提示

## 💡 使用技巧

### 高效分析技巧
1. **批量操作**: 可以先加载数据，然后依次分析多个用户
2. **对比分析**: 可以切换不同用户，对比他们的社交网络差异
3. **深度探索**: 结合多度人脉和可视化，深入理解社交结构

### 可视化优化技巧
1. **缩放查看**: 放大查看细节，缩小查看全局结构
2. **拖拽定位**: 将感兴趣的区域拖到中心查看
3. **颜色识别**: 通过颜色快速识别不同类型的关联

### 数据管理技巧
1. **备份数据**: 定期备份重要的数据文件
2. **格式验证**: 导入前验证CSV文件格式
3. **增量更新**: 可以分批次添加用户和关系
