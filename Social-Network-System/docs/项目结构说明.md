## 🔧 核心模块详细说明

### 1. 核心算法模块 (core/)

#### AdjacencyList.java
**功能**: 社交网络图结构实现
**主要方法**:
- `addUser(int userId)`: 添加用户节点
- `addFriendship(int user1, int user2)`: 添加好友关系
- `getFriends(int userId)`: 获取用户好友列表
- `getAllUsers()`: 获取所有用户
- `hasUser(int userId)`: 检查用户是否存在
- `getEdgeCount()`: 获取边数量

#### GraphAlgorithm.java
**功能**: 图算法实现
**主要方法**:
- `getFirstDegreeFriends(int userId)`: 一度人脉查询
- `getSecondDegreeFriends(int userId)`: 二度人脉查询
- `getMultiDegreeFriends(int userId, int degree)`: 多度人脉查询
- `calculateSocialDistance(int user1, int user2)`: 社交距离计算
- `getShortestPath(int user1, int user2)`: 最短路径查询

#### HashTable.java
**功能**: 哈希表数据结构（链地址法）
**主要方法**:
- `put(int key, User value)`: 插入键值对
- `get(int key)`: 获取用户信息
- `size()`: 获取元素数量
- `containsKey(int key)`: 检查键是否存在

#### RecommendationEngine.java
**功能**: 智能推荐算法
**主要方法**:
- `recommendUsers(int userId, int k)`: 推荐k个用户
- `calculateInterestSimilarity(User user1, User user2)`: 兴趣相似度计算
- `getCommonFriends(int user1, int user2)`: 获取共同好友

### 2. 数据处理模块 (data/)

#### DataLoader.java
**功能**: 数据文件加载器
**主要方法**:
- `loadFromFiles(String usersFile, String friendshipsFile, HashTable userTable, AdjacencyList graph)`: 从文件加载数据
- `parseUserLine(String line)`: 解析用户数据行
- `parseFriendshipLine(String line)`: 解析好友关系行

### 3. 图形界面模块 (gui/)

#### Launcher.java
**功能**: 界面启动器，提供新旧版本选择

#### SocialNetworkApp.java
**功能**: 新版主界面，现代化设计
**主要组件**:
- 顶部控制栏：标题、用户选择、数据加载
- 左侧功能菜单：六种分析功能
- 中间结果展示区：文本输出
- 底部可视化区域：图形显示

#### VisualizationPanel.java
**功能**: 社交网络可视化组件
**特色功能**:
- 鼠标滚轮缩放（0.5x-3.0x）
- 拖拽移动图形
- 点击节点查看详情
- 距离分级颜色显示

#### MainWindow.java
**功能**: 旧版界面，保留兼容性

### 4. 数据模型模块 (model/)

#### User.java
**功能**: 用户实体类
**属性**:
- `id`: 用户ID
- `name`: 用户姓名
- `interests`: 兴趣列表

## 🚀 编译和运行

### 编译所有文件
```bash
javac -d bin src/com/socialnetwork/**/*.java
```

### 运行启动器
```bash
java -cp bin com.socialnetwork.gui.Launcher
```

### 直接运行新版界面
```bash
java -cp bin com.socialnetwork.gui.SocialNetworkApp
```

## 🔄 模块依赖关系

```
SocialNetworkApp → Launcher (可选)
                → VisualizationPanel
                → GraphAlgorithm
                → RecommendationEngine
                → DataLoader
                
VisualizationPanel → GraphAlgorithm
                  → HashTable
                  
GraphAlgorithm → AdjacencyList
               
RecommendationEngine → AdjacencyList
                    → HashTable
                    
DataLoader → AdjacencyList
           → HashTable
           → User
```
